# Company Dashboard

A job search research tool built to explore AI-augmented design and full-stack deployment — designed and shipped by a product designer learning in public.

🔗 **[Live Demo](https://company-dashboard-kohl.vercel.app)**

---

## What it does

A dashboard for tracking and visualizing company research during a job search — helping surface signal across dozens of targets at once, rather than managing tabs and spreadsheets.

---

## How it was built

This project was an intentional experiment in AI-assisted design and development — not a shortcut, but a way to move fast and learn faster.

| Tool | How I used it |
|---|---|
| **Magic Patterns** | Prompted iterative UI design and critiqued layout decisions against IA principles |
| **Claude + GPT** | Debugged Supabase integration and TypeScript errors as a first-time Supabase user |
| **Vercel** | Deployed and iterated — 32 production deployments over the course of the project |

The goal wasn't to generate code and walk away. It was to stay in the driver's seat on design decisions while using AI to close the gap between design intent and working product.

---

## Stack

- **TypeScript** / React
- **Supabase** (database + auth)
- **Tailwind CSS**
- **Vite**
- **Vercel** (deployment)

## What I learned

- How to prompt AI tools for design critique, not just generation
- Supabase schema design and row-level security basics
- The feedback loop between deploying early and discovering what actually needs fixing
- Where AI assistance breaks down and judgment calls take over
